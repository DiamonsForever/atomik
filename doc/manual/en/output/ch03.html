Requirements

  • HTTP Server. Apache with mod_rewrite is a good choice.
  • PHP 5 or greater

Download

You can download Atomik from the Atomik website

Atomik is distributed under two forms:

  • An archive containg a full directory structure
  • A single file

You can create your own Atomik distribution online with Atomik Builder. It allows you to pack in plugins and pre-configure your installation online.

Installation

Using an archive

  1. Unzip the downloaded archive

  2. Upload or move Atomik files and folders into a directory available from the web.

  3. Code!

Using the all-in-one index.php

  1. Copy the index.php file to a directory available from the web

  2. Open a terminal and enter the following command: php index.php init

    You can also use the --htaccess option to create a .htaccess file: php index.php init --htaccess

  3. Code!
