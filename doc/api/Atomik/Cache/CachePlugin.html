Cache plugin

Cache the whole output of a request.

Specify which requests to cache using the cache_requests config key. cache_requests value must be an array where its keys are the request and its values the time in second the cached version will be used. Example:

array( 'index' => 60, 'view' => 3600 )

The /index request will be catched for 60 seconds and the /view request for one hour.

Other possible time value:

  1. = use default time
-1 = infinite

The cache is regenerated without taking in consideration time when the action or the template associated to the request are modified.
